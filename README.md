# IPL Data Analysis

## Project Overview

This project analyzes Indian Premier League (IPL) historical data using Python, Pandas, Matplotlib, and Seaborn. The goal is to explore team performance, player statistics, scoring trends, and match outcomes through data analysis and visualization.

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab
* GitHub

## Dataset

The dataset used in this project was obtained from Kaggle and contains IPL match-level and ball-by-ball data.

## Data Cleaning

The following data cleaning steps were performed:

* Handled missing values
* Removed unnecessary spaces and inconsistencies
* Converted columns to appropriate data types
* Merged multiple datasets using common keys
* Prepared data for analysis and visualization

## Analysis Performed

1. Toss Impact on Match Results
2. Top Run Scorers in IPL History
3. Top Six Hitters in IPL History
4. Top Wicket Takers in IPL History
5. Runs Scored by Season
6. Orange Cap Winners by Season
7. Most Fours Hit by a Player in Each Season
8. Cities Hosting the Most IPL Matches
9. Most Player of the Match Awards
10. Highest Team Scores in IPL History
11. Distribution of Runs per Ball

## Key Insights

* IPL scoring has increased significantly over the years.
* Winning the toss provides only a slight advantage.
* Virat Kohli recorded one of the greatest batting seasons in IPL history.
* Sunrisers Hyderabad dominate the list of highest team scores.
* AB de Villiers has won the most Player of the Match awards in the dataset.
* Most deliveries result in 0 or 1 run, while boundaries contribute significantly to team totals.

## Visualizations

The project includes:

* Bar Charts
* Line Charts
* Pie Charts
* Seaborn Distribution Plots

## Conclusion

This project demonstrates how data analytics can be applied to sports data to uncover meaningful insights. Using Python and data visualization libraries, IPL performance trends, player achievements, and match outcomes were analyzed effectively.

## Important Note

The dataset contains partial data for the 2026 IPL season. Therefore, analyses involving season-wise comparisons may show lower values for 2026 compared to previous seasons. To avoid misleading conclusions, the 2026 season was treated as incomplete data and was excluded from trend-based interpretations where appropriate.
